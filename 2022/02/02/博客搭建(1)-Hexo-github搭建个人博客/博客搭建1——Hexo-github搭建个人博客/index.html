下列教程是我按照bilibili网站中博客搭建教程进行归纳整理的，我的个人博客就是按照该流程进行操作搭建，其中个别部分做出了修改，为方便日后记忆，特此记录。

  写在开头：个人博客搭建不需要任何编程基础，只要按照流程进行即可，但由于版本或其他原因，在搭建过程中可能会遇到一些error，遇到困难不要急，冷静下来先想清楚自己遇到的是什么问题，如何描述，然后在诸如Google、CSDN中进行搜索，在这个过程中你会发现，社区环境非常友好，大家互帮互助，加入他们吧。
特别注明，这篇教程主要依据如下2个教程编写，为方便综合对照，文章标题与视频教程标题相同。

搭建仓库

  首先创建一个GitHub账号，创建完成后，进入账号主页面，点击左上角的new创建一个新的仓库。

进入编辑仓库页面后，编辑如下：

  1. 编辑仓库名字，统一格式：xxxgithub注册名 .github.io，此处xxx一定要为github注册名
  2. 点击Add a README file
  3. 点击Create a license创建新的仓库
